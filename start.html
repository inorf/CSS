Kaskadowe Arkusze Stylów CSS

CSS - Kaskadowe Arkusze Stylów (z ang. Cascading Style Sheets) jest to specjalny język stworzony w celu elastycznego zarządzania sposobem formatowania (wyglądem) elementów znajdujących się na stronach WWW.

Tło właściwości odpowiedzialne za formatowanie cech wyglądu tła w danym elemencie HTML
Marginesy właściwości odpowiedzialne za formatowanie wartości marginesów wewnętrznych i zewnętrznych w danym elemencie HTML
